## 5. Master Topic List (Full)

*Front-end / consumer features only. No DAX, Desktop, modeling, or Power Query.*

### Level 1 — Beginner (Getting Around) — 9 modules

**1.1 What Is the Power BI Service?** — Service vs. Desktop, sign-in at app.powerbi.com, license types (Free / Pro / PPU).

**1.2 Navigating the Home Screen** — Home layout (Recents, Favorites, Featured), left nav (Home, Browse, Data Hub, Apps, Metrics, Learn), global search, notification center.

**1.3 Understanding Content Types** — Dashboard vs. Report vs. App; Workspace from a consumer view; Semantic Model/Dataset high-level.

**1.4 Viewing & Interacting with Reports** — Page tabs, Reading vs. Edit mode, tooltips, cross-filter/highlight, visual types, sorting, Show as Table, Focus mode, Full-screen, Spotlight.

**1.5 Using Slicers & Filters** — Slicers, Filters pane (page/report/visual), clearing filters, filter-applied indicators.

**1.6 Viewing Dashboards** — Dashboard vs. report, tile click-through, TV/full-screen mode, dashboard Q&A.

**1.7 Using Power BI Apps** — What Apps are, installing from Apps section, navigating inside, Org vs. Workspace Apps.

**1.8 Favorites & Recents** — Marking favorites, accessing favorites, Recents, "Shared with me" under Browse.

**1.9 Power BI Mobile App Basics** — Install iOS/Android, sign-in, viewing reports/dashboards, mobile-optimized layouts.

### Level 2 — Intermediate (Getting Productive) — 14 modules

**2.1 Drill Down, Drill Up & Drill Through** — Hierarchical drill, expand/collapse all on matrix, drill-through, back button.

**2.2 Personal Bookmarks** — Create/update/delete, default personal bookmark, report vs. personal bookmarks.

**2.3 Persistent Filters** — Session memory, reset to default.

**2.4 Exporting Data & Visuals** — Export to Excel/CSV, PDF, PowerPoint, print, copy visual as image.

**2.5 Analyze in Excel** — Live dataset connection, opening Excel workbook, PivotTables, refresh.

**2.6 Subscribing to Reports & Dashboards** — Email subscriptions, schedules, page/bookmark subscriptions, manage/delete.

**2.7 Data Alerts (Dashboards)** — Alerts on gauge/KPI/card tiles, thresholds, notifications, manage.

**2.8 Commenting & Collaboration** — Comments, @mentions, threads, bookmark-linked comments.

**2.9 Sharing Content** — Share links, specific people vs. org, share with filters, permissions (view vs. reshare).

**2.10 Power BI in Microsoft Teams** — Power BI tab, embedding in channels/chat, viewing in Teams, chat about reports.

**2.11 Q&A (Natural Language Queries)** — Dashboard Q&A, Q&A visual in reports, phrasing tips, pinning results.

**2.12 Quick Insights** — Run on a tile, insight types (trends/outliers/correlations), when to use.

**2.13 View Related Content** — Related dashboards/reports/datasets, connections, "View related" navigation.

**2.14 Copy Visual as Image** — Right-click/menu copy, paste to email/docs/slides, copy vs. export.

### Level 3 — Advanced (Power User) — 12 modules

**3.1 Copilot in Power BI** — Consumer Copilot, Copilot pane, standalone Copilot, summarize pages, NL data questions, limitations, mobile.

**3.2 Goals & Scorecards (Metrics)** — Scorecards, goal status, manual check-ins, following goals, connecting to report data.

**3.3 Featured Tables & the Data Hub** — Endorsed/certified datasets, Featured Tables in Excel, endorsement labels, Data Hub vs. workspaces.

**3.4 Power BI Add-in for PowerPoint** — Live embed, interactive filtering in presentations, live vs. static, permissions.

**3.5 Power BI Add-in for Excel (Live Connection)** — Excel-direct dataset connection, live-data reports, refresh.

**3.6 Translytical Task Flows** — Actions from reports (approve orders, update records), triggering workflows.

**3.7 Content Discovery & Governance (Consumer Side)** — Lineage view, impact analysis, sensitivity labels, Promoted vs. Certified.

**3.8 Advanced Sharing & Distribution** — Subscriptions for others, RLS for consumers, SharePoint embed, secure embed links.

**3.9 Accessibility Features** — Keyboard nav, screen reader, high-contrast, alt text/tab order, focus mode.

**3.10 Notifications, Activity & Troubleshooting** — Notification center, Get Help, access denied messages, requesting access.

**3.11 Mobile Advanced Features** — Favorites/sharing from mobile, annotate/share snapshots, QR scan, mobile alerts.

**3.12 Personalizing Visuals (If Enabled)** — Personalize this visual, change type/axis/values, save to personal bookmarks, author-enabled limitation.
